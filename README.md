## 第三章 搭建一个轻量级的 Java Web框架

### 3.1 目标

​		MVC(model-view-controller)是一种常用的设计模式，可以使用这个模式将应用程序进行解耦。

框架功能:

1. 减少Servlet的数量，业务交由设计的Controller来处理
2. 实现依赖注入，自动获取Controller需要的Service

需要技能

- 加载配置文件
- 搭建简单的框架
- 实现简单的IoC
- 指定加载类
- 初始化框架
